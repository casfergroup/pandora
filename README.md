## Creating and Editing Pages

### Code
The code part of the markdown needs to be opened with the following style:

```
language_here {linenos=inline style=base16-snazzy}
```

Refer to Hugo's [Syntax Highlighting](https://gohugo.io/content-management/syntax-highlighting/) and [Syntax Styles](https://gohugo.io/quick-reference/syntax-highlighting-styles/) for more information.
